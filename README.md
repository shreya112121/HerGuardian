# HerGuardian 🛡️

HerGuardian is a Women's Safety Platform built using Java, Spring Boot, Spring Security, Hibernate, and MySQL.

## Features
- One-tap SOS Alert
- Real-time GPS Location Sharing
- Emergency Contacts Management
- Live Walk Tracking
- Nearby Hospitals, Police Stations & Shelters
- JWT Authentication
- Responsive UI

## Tech Stack
- Java
- Spring Boot
- Spring Security
- Hibernate
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Project Highlights

- Real-time GPS location tracking
- SMS alerts using Fast2SMS API
- JWT-based Authentication & Authorization
- Emergency Contacts Management
- Nearby Hospitals, Police Stations & Shelters
- Walk Session Monitoring with Safety Timer
